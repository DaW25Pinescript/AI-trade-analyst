# Weekly Review Prompt Template (V3)

Use this template to summarize a week's tickets and AARs with deterministic metrics.

## Inputs
- Closed tickets in period
- AAR records in period
- Metrics output (`app/scripts/metrics/metrics_engine.js`)

## Prompt skeleton
1. Weekly performance summary
2. Pattern breakdown (wins/losses)
3. Calibration and confidence drift
4. Next-week adjustments and guardrails
