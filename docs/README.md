## Documentation Doctrine

- `docs/AI_TradeAnalyst_Progress.md` is the **canonical** progress/status document for current phase, next actions, debt tracking, and milestone history.
- Architecture/spec/design/runbook documentation exists to **support** implementation and operations around that canonical plan.
- Historical snapshots, superseded plans, and old audits belong in `docs/archive/` and must not compete with the canonical progress hub.
